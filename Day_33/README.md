# Simple Login System

This is a simple login system built with Python and Streamlit. It allows users to register and log in using a username and password. User credentials are stored in a local JSON file.

## Features

- User registration with password hashing (using bcrypt)
- User login with password verification
- Session management to keep users logged in
- Simple and clean UI using Streamlit

## How to Run

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Install the dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Streamlit application:**
   ```bash
   streamlit run app.py
   ```

4. Open your browser and go to the local URL provided by Streamlit (usually `http://localhost:8501`).
